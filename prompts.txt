【JavaScript 必須實現的功能】
- const passInfo = tripData.passes;
- closeAllModals()：關閉所有 modal 並收合所有 day-group
- toggleVjwModal()：專門開關 vjw-modal
- toggleDay(event)：允許多個 day 展開（非強制單一展開），切換當前 group 的 'active' 類別，並將 .toggle 文字從 '▼' 改為 '▲' 或反之
- openModal(e)：點擊 pass-tag（透過 data-modal）開啟通用 modal 顯示 pass 資訊，包含 title、info、benefits/save
- openMap(lat, lng, name)：產生 Google Maps 連結，使用 https://maps.google.com/?q=${encodedQuery}&ll=${lat},${lng}，name 若無「站/機場/神社/寺/市/市場/橋」則自動補「 北九州」
- 滾動 >300px 顯示 FAB
- render() 函式：根據 tripData.days 動態產生所有 day-group 與 spot HTML
  - spot 若有 pass/cost/save → 產生對應 tag 如 <span class="tag pass">${s.pass} 涵蓋 <s>¥${s.cost}</s> <span class="tag save">省 ¥${s.save}</span></span>
  - spot 若有 pass 但無 cost/save → 產生 <span class="tag pass">${s.pass} 涵蓋</span>
  - spot 若有 lat/lng → 產生「地圖」按鈕 <button class="map" onclick="event.stopPropagation();openMap(${s.lat},${s.lng},'${s.title}')">地圖</button>
  - spot 可點擊打勾（class done + 勾勾 + 刪除線）
  - spot class 加入 type 如 class="spot ${s.type}"
  - day-header 結構：包含 <div class="day-title">${day.d} ${day.title}</div> 和 <div style="font-size:12px;color:var(--sub)">${day.date}</div>
  - 最後呼叫 render()，並為 .pass-tag[data-modal] 綁定 click 事件到 openModal
